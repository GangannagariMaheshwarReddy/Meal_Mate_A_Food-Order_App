MealMate is a full-stack **Restaurant Management & Food Ordering System** built with **Django**. It provides a simple and intuitive interface for **Admins** to manage restaurants and menus, and for **Customers** to browse restaurants, view menus, and place orders.  

This project is ideal for learning **Django CRUD operations, templating, and web app design**.

---

## Features

### Admin Features
- Add, Update, and Delete Restaurants.
- Add, Update, and Delete Menu Items.
- Manage restaurant ratings and cuisines.

### Customer Features
- Browse restaurants and menus.
- View item images, descriptions, and prices.
- Add items to cart and checkout (basic simulation).
- User-friendly interface with modern styling.

---

## Tech Stack

- **Backend:** Django 5.2, Python 3.13  
- **Frontend:** HTML5, CSS3, Django Templates  
- **Database:** SQLite (default, can switch to MySQL)  
- **Tools:** VS Code, Git, Browser for testing
